**Современный(почти) Android-мессенджер, созданный для свободного и удобного общения.** 

## О проекте

Mayas — это экспериментальный прикол, написанный на Kotlin с использованием Jetpack Compose.
Цель проекта — поугарать... А еще сделать среду где вы сможете общаться со своими близкими

## Возможности

* Общение в реальном времени ( почти ) 
* Крутая реклама 
* Крутой маяс+ 
* Поддержка медиафайлов ( Скоро! )
* Быстрая и лёгкая работа приложения ( Если не заходить )
* Там нет мема сикс севен

## Конфиденциальность

Mayas создаётся с упором на приватность пользователей.

* Ваши данные не продаются третьим лицам ( А только передаются серверу )
* Минимальный сбор информации ( Если он там вообще есть )
* Безопасное хранение пользовательских данных
* Упор на комфортное и безопасное общение

## Философия проекта

Общение должно быть **_свободным_**

- Без VPN.
  
   ###  **И**
  
- Без лишних ограничений.

Мы вам не навяжем его в отличии от ... А впрочем не важно)
